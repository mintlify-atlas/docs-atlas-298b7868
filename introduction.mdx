## What is NexPhisher?

NexPhisher is a Bash-based phishing framework that automates the creation and deployment of credential-harvesting pages. It replicates the login pages of 30+ major platforms and serves them via various port-forwarding services to make them accessible over the internet.

The tool is designed for use on Linux systems and Android devices running Termux. It features an interactive terminal menu, real-time credential capture, and victim IP logging.

## How it works

NexPhisher operates in three stages:

1. **Template selection** — You choose a target platform from the interactive menu. NexPhisher copies the corresponding HTML/PHP login page template to a local web server directory.

2. **Tunnel setup** — A PHP built-in web server is started locally. A tunnel service (Ngrok, Serveo, etc.) creates a public URL that forwards to the local server.

3. **Credential capture** — When a victim submits the fake login form, the credentials are written to `usernames.txt` and the victim's IP is written to `ip.txt` in the server directory. NexPhisher's polling loop detects these files and displays the data in the terminal.
